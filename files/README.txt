To add your logo:

1. Create or find a logo image for your course/institution
2. Recommended size: 512x512 pixels (PNG format)
3. Save it as `favicon-512.png` in this folder
4. Optionally create additional sizes:
   - favicon-32x32.png
   - favicon-16x16.png

The logo will appear in:
- Browser tab (favicon)
- Navigation bar
- Home page

Simple logo creation options:
- Canva (https://www.canva.com/)
- GIMP (free, open-source)
- Adobe Photoshop
- Use your institution's official logo

For now, the site will work without the logo, but adding one makes it look more professional!
